Build raftkeeper without tcmalloc. cmake -D ENABLE_TCMALLOC=0

Copy raftkeeper binary to your server.
scp programs/raftkeeper server:~

ssh to your server

Stop raftkeeper:
sudo service raftkeeper-server stop

Run raftkeeper with heap profiler from the terminal:
sudo -u raftkeeper LD_PRELOAD=/usr/lib/libtcmalloc.so HEAPPROFILE=/var/log/raftkeeper-server/heap.hprof ./raftkeeper server --config /etc/raftkeeper-server/config.xml

Profiles will appear in /var/log/raftkeeper-server/
